# SkyWatch 🌤️

SkyWatch is a **real-time weather web application** built using **HTML, CSS, and JavaScript**.
It automatically detects your current location and displays **live weather conditions, a 6-day forecast, and weather data for cities worldwide**.

---

## 🚀 Features

* 🌍 **Auto Location Weather** – Detects your location and shows current weather instantly
* 🔄 **Temperature Toggle** – Switch between **Celsius (°C)** and **Fahrenheit (°F)**
* 📅 **6-Day Forecast** – Scrollable forecast cards with weather conditions
* 🔎 **City Search** – Search weather for any city worldwide
* 🌎 **Explore Page** – View weather for popular cities and add new ones
* 📱 **Responsive Design** – Works across desktop and mobile devices

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```
git clone https://github.com/nayan-khantal/skywatch.git
```

### 2️⃣ Navigate into the project

```
cd skywatch
```

### 3️⃣ Add your OpenWeather API key

Open the following files:

```
js/main.js  
js/search.js  
js/world.js
```

Replace the API key with your own:

```
let apiKey = "YOUR_API_KEY_HERE";
```

You can get a free API key from:
https://home.openweathermap.org/api_keys

⚠️ New API keys may take **up to 2 hours to activate**.

### 4️⃣ Run the project

Simply open **index.html** in your browser.
